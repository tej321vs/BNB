# AI-Powered Student Career Pathway Navigator

## Project Description
The AI-Powered Student Career Pathway Navigator is a web application designed to help students identify suitable career paths based on their skills and interests. It combines machine learning, interactive dashboards, and a user-friendly interface to provide personalized guidance.

## Features
- **Skill-Based Analysis:** Inputs student skills to recommend career paths.
- **Interactive Visualizations:** Uses Plotly to display data-driven insights.
- **Cloud Deployment:** Hosted and scalable on Google Cloud Run.
- **AI Integration:** Leverages machine learning models for personalized recommendations.

## Tech Stack
- **Backend:** Python, Flask
- **Frontend:** HTML/CSS, Plotly
- **Deployment:** Google Cloud Run
- **Data Processing:** Pandas

## How to Run
1. Clone the repository.
2. Install requirements using `pip install -r requirements.txt`.
3. Run the app using `python app.py`.
